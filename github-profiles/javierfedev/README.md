| Comando | Descripción  |
|----------|----------|
| ls     | Listar el contenido de una ubicación  |
| mkdir    | Crear un directorio nuevo  |
| cp   | Copiar elementos a una nueva ubicación   |
| mv   | Mover elementos a otra ubicación   |
| cd   | Cambiar de directorio  |
